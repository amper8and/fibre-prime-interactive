# MTN Fibre Prime - Quick Start Guide

## 🚀 Get Started in 3 Steps

### 1. Clone Repository
```bash
git clone https://github.com/amper8and/fibre-prime-interactive.git
cd fibre-prime-interactive
```

### 2. Install & Run
```bash
npm install
npm run dev
```

### 3. Open Browser
Visit: http://localhost:3000

---

## 📱 Application Structure

### Main Pages
- **/** - Landing page
- **/experience** - Interactive home (main feature)
- **/marketplace** - Product catalog
- **/plans** - Fibre plans
- **/about** - Platform info

### Key Features
- 7 explorable rooms
- 25+ interactive devices
- Bundle builder
- Fibre plan comparison
- Mobile responsive

## 🎯 Quick Deploy

### Vercel (1 minute)
```bash
npm i -g vercel
vercel
```

### Netlify (2 minutes)
```bash
npm run build
npm i -g netlify-cli
netlify deploy --prod
```
